The old notebook is now deprecated.

Starting from version 9.0, Sage is using Python3 by default. The old Sage notebook only works for Sage using Python2 and is now deprecated. The old Sage notebook is still available as an optional package, working only under Python2 and that you need to install using sage -i sagenb.
